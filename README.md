## Common Questions

**How do I install the trainer?**  
Extract the downloaded archive to any folder, run the executable after launching Dead by Daylight, and follow the on-screen prompts.

**Will I receive updates automatically?**  
If enabled in settings, the trainer will check for new versions on startup and prompt you to download them.

**Can I customize the hotkeys?**  
Yes. Open the configuration panel with F1 and assign new keybinds for each feature.

**Does it work with every game update?**  
We strive to maintain compatibility, but occasional game patches may require a trainer update. Check the download page for the latest version.

**Where are my settings stored?**  
Configuration files are saved in the same directory as the trainer executable in a `config.ini` file.
